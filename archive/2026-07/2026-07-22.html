存储芯片领涨反弹·英特尔财报前抢跑·科创半导体集体涨停

一、盘后总结

美东时间7月21日（周二），美股三大指数集体收涨，一举结束此前的连续调整：道琼斯工业平均指数涨385.38点，涨幅0.74%，报52,224.64点；标普500指数涨0.89%，报7,509.20点；纳斯达克综合指数涨1.29%，报25,837.21点，涨幅居三大指数之首；罗素2000小盘股指数涨22.59点，涨幅0.77%，报2,965.02点；VIX恐慌指数大幅回落5.95%至17.54（TheStreet、The Motley Fool）。费城半导体指数（SOX）单日跳涨约5.5%，是拉动大盘的最主要引擎，资金明显从此前的防御/周期板块（医疗保健XLV、材料XLB、工业XLI当日分别下跌1.2%、0.9%、0.8%）切换回成长科技，尤其是半导体链内部的存储子板块。

本轮反弹呈现清晰的"存储领涨、设备跟随、逻辑芯片补涨"结构：iShares半导体ETF（SOXX）盘中一度上涨4%至547美元，NVIDIA、Marvell、台积电、SK海力士、Nebius等均同步走强，显示反弹并非单一个股驱动，而是全链条的risk-on。亚太市场对此已提前定价——日经225指数收涨3.3%，韩国Kospi指数收涨3.6%，台湾加权指数收涨4.2%，为隔夜美股半导体股的走强提供了先行信号（24/7 Wall St.）。需要指出的是，本轮反弹距离7月中旬的深度回撤（SOXX过去一个月一度下跌18%，英特尔股价一个月内下跌28%）尚有明显距离，多篇报道将其定性为"修复而非突破"。

个股层面，半导体链大幅跑赢M7：美光科技（Micron）单日涨幅在12%至12.6%之间（不同信源口径略有差异，两个口径并列，参见下方个股榜单）、SanDisk涨约14%，而M7阵营中NVIDIA仅上涨2%，其余成分股当日波动相对温和，未见显著异动报道。云厂商内部同样分化明显：受"AI是否会侵蚀企业软件"担忧影响，ServiceNow、HubSpot等SaaS股走弱，而AI基础设施类云服务商（CoreWeave、Nebius、IREN）此前因Meta自建云的消息持续承压，估值从年内高点大幅回落，当日跟随硬件链有所企稳，但基本面仍有待验证。

三、AI/半导体全产业链

AI模型/应用与资本开支

  • Meta / Microsoft / Alphabet / Amazon：四家公司2026年合计资本开支指引升至约7,250亿美元，较上年410亿美元的口径同比增长约77%；其中Amazon约2,000亿美元、Alphabet 1,750亿-1,850亿美元、Microsoft 1,900亿美元（高于市场此前1,520亿美元的平均预期）、Meta全年指引上调100亿美元至超过1,450亿美元 → 验证点是三季度财报中各家资本开支的实际落地节奏，以及Microsoft CFO所提及的"约250亿美元资本开支增量来自内存芯片与零部件成本上升"这一说法能否在后续财报中得到量化验证（Tom's Hardware）。

CSP/云资本开支

  • Microsoft：管理层表示公司在GPU、CPU、存储基础设施建设上仍将维持"产能受限"状态，预计这一状态至少持续到2026年底 → 验证点是三季度财报电话会中对数据中心利用率与积压订单（backlog）的披露。
  • Meta：将资本开支上调归因于内存芯片涨价，以及数据中心用地、电力和技术人才的竞争加剧 → 验证点是四季度指引是否进一步上修，以及自建云（对Nebius/CoreWeave/IREN等"新云"构成的替代压力）的具体项目披露。

AI云/数据中心运营商

  • CoreWeave：公司营收积压（backlog）约994亿美元，其中包含与Meta的210亿美元合约，2026年资本开支指引310亿-350亿美元；股价此前已从年内高点186美元大幅回落 → 机制层：市场担忧的核心是新云厂商的资本开支强度与订单兑现速度是否匹配，而非需求本身；验证点是后续季度backlog向实际收入转化的比例，以及自由现金流拐点时间表。
  • IREN：目标到2026年底实现37亿美元ARR，已锁定5吉瓦全球电力资源，并与NVIDIA签订34亿美元云合约；股价一度较年内高点回落52%，做空比例达16% → 验证点是电力资源能否按期转化为可计费算力容量。
  • Nebius：股价此前在Meta自建云消息冲击下从高点大幅回落，做空比例约20% → 验证点是后续大客户合约（尤其是与超大规模厂商的直接合作）能否对冲"新云被自建云替代"的担忧（24/7 Wall St.）。

GPU/CPU/ASIC

  • AMD：将于7月22-23日召开"Advancing AI"大会，为本周关键催化剂；一季度财报显示营收102.5亿美元，同比增长38%，CEO Lisa Su将增长归因于AI基础设施需求加速、MI450系列产品放量，以及与Meta的6吉瓦部署合作 → 验证点是大会上是否披露新的超大规模客户或MI450具体出货量指引。
  • Broadcom：最新一季AI半导体收入108亿美元，同比增长143%，并指引2026财年三季度AI半导体收入约160亿美元；与苹果7月8日签署的300亿美元多年期芯片协议（至2031年）仍是持续性利好 → 验证点是三季度AI收入指引能否兑现，以及苹果协议后续订单细节披露。
  • NVIDIA：当日股价仅上涨2%，涨幅弱于存储与部分逻辑芯片同业 → 机制层：这更多反映的是资金在半导体链内部的再平衡，而非对NVIDIA基本面的重新定价；验证点是后续季度数据中心收入增速能否维持。

HBM/DRAM/NAND/SSD/HDD

  • DRAM/NAND合约价：TrendForce预测三季度DRAM合约价环比上涨13%-18%、NAND合约价环比上涨10%-15%，较二季度约60%的环比涨幅明显放缓；但存储模组厂商ADATA给出的口径更激进，称收到供应商三季度DRAM涨价20%-30%、NAND涨价35%-40%的通知——两个口径并列，反映产业链内不同环节对涨价幅度的认知存在分歧 → 验证点是8月各大存储厂商正式合约价的落地水平。
  • SK海力士 / 三星电子：据行业分析，SK海力士在HBM市场份额约50%-55%，三星约35%-40%；两家公司2026年策略出现分化——SK海力士被指优先保障DDR5的高利润率而放缓HBM4放量节奏，三星则在HBM4良率问题后加速追赶 → 验证点是三季度末双方HBM4量产良率与NVIDIA新一代16-Hi HBM认证结果（Silicon Analysts）。此外，Samsung、SK海力士、Micron三家近期同时面临一起新的DRAM价格垄断调查，需关注监管进展对合约定价机制的潜在影响。
  • Micron / SanDisk：当日分别上涨12%~12.6%、14%，是本轮反弹中弹性最大的两只个股，直接受益于NAND/HBM涨价与"供应短缺持续至本十年末"的行业判断 → 验证点是两家公司下一季度财报中存储业务毛利率的实际改善幅度，警惕单日涨幅只反映交易情绪、不构成基本面证据本身。

晶圆代工与先进封装

  • 台积电(TSM)：7月21日股价上涨3.29%；CoWoS先进封装产能瓶颈持续收窄，据报道供需缺口有望从目前约20%收窄至2026年底的10%，月产能有望扩大至12万-14万片 → 机制层：CoWoS产能扩张是AI芯片实际出货量能否兑现订单预期的核心变量，产能单元数不等于实际出货量，需与下游GPU/ASIC厂商的排产计划交叉验证；验证点是台积电三季度财报电话会对CoWoS产能利用率与2027年扩产计划的表述。
  • ASML：7月15日公布二季度业绩，净销售额93亿欧元，净利润29亿欧元，上调2026全年销售额指引至430亿-450亿欧元区间，装机基础管理收入超计划约3亿欧元，主要来自生产力与软件升级 → 验证点是下半年EUV设备订单出货节奏（ASML 6-K文件）。

半导体设备/测试

  • 晶圆制造设备（WFE）市场：摩根士丹利分析师Shane Brett将2026年WFE支出预测由此前1,360亿美元上调至1,430亿美元，同比增幅达23%，并将2027年预测由1,610亿美元上调至1,820亿美元，认为设备环节正在承接AI基建价值链中更多的增量，应用材料（AMAT）被列为美国设备板块首选 → 验证点是应用材料、拉姆研究（7月29日）、科磊（7月30日）即将发布的财报指引是否与该上修预测方向一致（该研报具体发布日期未能进一步核实，作为背景参考）。
  • Teradyne：当日上涨8.69%，收报362.78美元，跟随设备板块整体反弹，未见个股专属催化剂报道 → 验证点是暂无，事件驱动栏如实标注"暂无"。

PCB/载板/光通信/电源

  • ServiceNow (NOW)：收报102.06美元，跌2.5%，市场预期其即将发布的二季度营收39.2亿美元、每股收益0.86美元，年内跌幅已达31.65%（相较141.64美元的平均目标价隐含约35%上行空间）→ 机制层：此前IBM曾发布预警称企业客户正将预算从软件转向AI硬件采购，引发SaaS板块整体抛售，ServiceNow与Salesforce均受波及；验证点是财报中订阅收入留存率与AI相关产品（如Now Assist）的采纳数据。

九、跨市场提示
